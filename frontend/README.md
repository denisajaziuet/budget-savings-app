# Budget Savings Frontend

Aplikacioni frontend është ndërtuar me React, Redux Toolkit dhe RTK Query.

## Si të niset
1. Hyni te folderi `frontend`
2. Kopjoni `.env.example` në `.env`
3. Ekzekutoni:
   ```bash
   npm install
   npm run dev
   ```

## Rrugët
- `/` - Paneli
- `/hyrje` - Hyrje
- `/regjistrohu` - Regjistrim
- `/shpenzimet` - Menaxhimi i shpenzimeve
